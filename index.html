Tags HTML usadas:

<!DOCTYPE html> para indicar que é um documento html, usado em todas as páginas html.

<html></html> para indicar o inicio e fim do documento html.

<head></head> local onde inseri informações sobre o documento html tais como o título que aparece no separador de navegação, a inclusão de ficheiros css e javascript, o "favicon", entre outras informações importantes.

<meta> para incluir metadados. Com esta tag defini a codificação e a apresentação da página na largura total segundo o dispositivo usado bem como o zoom inicial (tamanho real-100%).

<link> foi usada para poder definir o favicon e a inclusão do ficheiro css.

<script></script> foi usada para incluir o ficheiro com o código javascript.

<title></title> usei para definir o título "LSSWeb - E-folio B" que aparece no separador do navegador.

O código definido na tag <head></head> não é apresentado ao utilizador.

<body></body> é onde fica o código html que vai "formatar" o que é apresentado ao utilizador.

<div></div> usei ao longo de todo o website para definir o layout com secções/"containers". Por exemplo para colocar o logótipo e o menu.

<a></a>usei para criar links. Por exemplo para definir nos itens do menu, o destino quando o utilizador clica numa opção do menu.

<h1></h1> para criar um título principal com a frase "Tee Time".

<h2></h2> para criar um título secundário com a frase "Descubra o mundo do golfe.".

<h3></h3> para criar um título mais específico com a frase "Bem-vindo ao Tee Time!".

<p></p> para criar um parágrafo com a frase "Para se inspirar."

<img></img>para inserir a imagem do logótipo armazenada em "imagens/geral/logo_tee_time.png".

<b></b> para colocar a negrito a frase "Tags HTML usadas:".

<br> para mudar de linha sem inserir espaço extra ente linhas com a frase "Laboratório de Sistemas e Serviços Web | e-fólio B".

Usei &lt; &gt; para poder escrever as tags de maneira a ser apresentado como texto e não serem interpretadas como código html.

Tags CSS usadas:

Usei @charset "utf-8" para definir a codificação de caracteres correta para o documento. Isso garante que os caracteres especiais sejam exibidos corretamente em diferentes navegadores e sistemas operacionais.

O "." permite seleccionar elementos a partir de um nome de classe, como exemplo ".cabecalho".

O "#" permite seleccionar elementos a partir de um nome de id, como exemplo "#imagens-thumbs".

Usei "html" para por exemplo, definir em toda a extensão do documento, a propriedade de fontes a usar.

"scroll-behavior:smooth" permite um deslocamento suave na página ao clicar no item de menu que vai para um destino na própria página, como exemplo clicar no "SOBRE".

O"body" seleciona todo o conteúdo visível da página e nele defini, por exemplo, que tem 100% de altura.

O "background-image" define a imagem de fundo que escolhi como sendo "../imagens/geral/golfe_background.jpg", o "background-attachment: fixed" para a imagem de fundo ficar fixa, "background-repeat: no-repeat" para não repetir a imagem de fundo, "background-size" Defini imagem à largura total com 100% e para não ficar distorcida a altura em "auto".

"width" usei no "body" para definir largura total (100%) ajustada ao dispositivo.

"margin" define a margem de um elemento, como exemplo no body tem margem 0.

"border" define o contorno dos elementos, neste caso usei no body com valor 0.

"font-family" define a família de fontes a utilizar nos navegadores.

Usei "height" para definir altura fixa, como exemplo 80px para o cabeçalho.

"background" usado em vários elementos para escolher a cor de fundo, como exemplo, no cabeçalho.

"display" ficou com valor "flex" (usei no elemento ".titulo-subtitulo-primario") para atribuir ao elemento a propriedade de ser flexível para os elementos dentro deste poderem se adaptar a tamanhos de ecrã diferentes dos vários dispositivos.

"flex-direction" usado no elemento ".titulo-subtitulo-primario" para indicar o modo de distribuição dos elementos.

"justify-content" é a maneira de organizar o espaço disponível para os elementos(usei no elemento ".rodape").

"align-items" serve para alinhar os elementos, como foi usado no elemento ".titulo-subtitulo-secundario".

"position" usado no elemento ".cabecalho" para fixar no topo da página.

"z-index" foi usado para que a barra (cabeçalho) no topo da página ficasse sempre à frente de todo o conteúdo.

Usei o "box-shadow" no elemento ".cabecalho" para adicionar um efeito de sombreado.

"text-align" utilizado para centrar o texto no elemento ".titulo-subtitulo-secundario".

Para definir a cor do texto presente no elemento ".titulo-subtitulo-secundario" usei "color".

"border-radius" usei para arredondar os cantos do elemento "#div-a-index-esq-conteudo".

Com "padding" defini em "#div-a-index-esq-conteudo" espaçamento entre o conteúdo e o limite do elemento. Também posso indicar individualmente caso necessário, se quero no topo (padding-top),fundo (padding-bottom),à esquerda (padding-left) e à direita (padding-right).

Usei "line-height" para alterar o espaçamento entre linhas do elemento ".espaco-linhas".

A tag "cursor" usei para indicar que as imagens miniaturas são elementos clicáveis. Elementos ".thumb".

Usei "box-sizing" no elemento ".div-1-coluna" para indicar que o tamanho dessa "div" inclui o "padding" definido, os "borders" e o conteúdo.

Usei "object-fit" no elemento ".thumb" para a imagem ocupar o tamanho todo do elemento.

Usei "text-decoration" no menu (".menu a") nos elementos "a" para não mostrar o sublinhado.

"transition" foi usado para definir a duração do efeito de transição nos links do menu.

Usei "font-size" no elemento ".legenda" para definir o tamanho da fonte.

Usei "font-weight" para definir o elemento ".nome-site" ("Tee Time") em negrito.

No caso do elemento ser seleccionado por classe, as propriedades aplicam-se a todos os elementos dessa classe. No caso de ser seleccionado por id, as propriedades apenas se aplicam a esse elemento pois o id é único.
